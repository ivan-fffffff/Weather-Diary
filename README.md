# Weather-Diary
Автор: Долгов Иван  
Краткое описание программы: данная программа предназначена для добавления, просматривания и фильтрации записи о погоде. Оно помогает вести хронологическую запись метеорологических наблюдений, анализировать их и сохранять для долгосрочного использовании.  
Инструкция по запуску:
Клонируйте репозиторий с GitHub.  
Запустите скрипт weather_diary.py.  
Добавляйте наблюдения, фильтруйте их, сохраняйте данные — всё в одном окне.
