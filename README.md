# Weather-Diary
Автор: Долгов Иван  
Описание программы: данная программа предназначена для добавления, просматривания и фильтрации записи о погоде. Оно помогает вести хронологическую запись метеорологических наблюдений, анализировать их и сохранять для долгосрочного использовании.  
как запустить программу:
Клонируйте репозиторий с GitHub.  
Запустите скрипт weather_diary.py.  
Добавляйте наблюдения, фильтруйте их, сохраняйте данные — всё в одном окне.
